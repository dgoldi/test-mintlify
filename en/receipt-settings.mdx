You create receipt configurations, define which receipt types are printed, and set the print order.

---

## Prerequisites

- Access to the **Admin App** with **Admin rights**
- Permission for the **Settings** section

---

## 1. Create a new receipt setting

1. Open the **Admin App**.
2. Click **Settings** in the sidebar.
3. Click **Receipt Settings**.
4. Click **Create Receipt**.

![Receipt settings overview](/images/en/beleg-einstellungen-liste.png)

5. Enter a **name** for the setting.
6. Enable or disable the **checkboxes** for the desired receipt types.
7. Change the **print order** using the arrow buttons.
8. Click **Save**.

### Available receipt types

| Type | Description |
|------|-------------|
| **Transaction receipt** | Standard register receipt. |
| **Card payment receipt** | Receipt for card payments (customer/store). |
| **Transaction receipt with signature** | Receipt with signature field. |
| **Gift card** | Gift card receipt. |
| **Service receipt** | Service receipt. |

---

## 2. Add or remove receipt types

- Click the **plus icon** to add a new receipt type.
- Click the **trash icon** to remove a receipt type.

---

## Frequently asked questions

**Can I change the print order?**
Yes. Use the arrow buttons to adjust the order of receipts.

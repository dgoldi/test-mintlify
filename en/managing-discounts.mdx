---

## Prerequisites

- Access to the **Admin App** with **Admin rights**
- Permission for the **Discounts** section

---

## 1. Create a new discount

1. Open the **Admin App**.
2. Click **Discounts** in the sidebar.
3. Click **Add Discount**.

![Discounts list with Add Discount button](/images/en/rabatte-liste.png)

4. Fill in the following fields:

| Field | Description |
|-------|-------------|
| **Name** | Designation of the discount. |
| **Type** | Type of discount (see discount types below). |
| **Percentage** or **Amount** | Discount value as percentage or fixed amount. |
| **Code** | Optional barcode for the discount. |
| **Single use** | When enabled, the discount can only be redeemed once. |
| **Active from / to** | Optional validity period. |
| **Customer** | Optional assignment to a specific customer. |
| **Redeemable Stores** | Optional restriction to specific stores. |

5. Click **Save**.

![Discount form with fields](/images/en/rabatt-formular.png)

---

## 2. Discount types

| Type | Description |
|------|-------------|
| **Article Discount** | Applied to the last scanned article. Repeatable per article. |
| **New Price** | Sets a new price on the last scanned article. |
| **Transaction Discount** | Applied at the end of the transaction. Not distributed across individual items. |
| **Distributed Transaction Discount** | Distributed across all items. Adjusts automatically when new articles are added. |
| **Individual Article Discount** | Applied once to all items. Can be adjusted per article afterwards. |

---

## 3. Mark a discount as favorite

Favorites appear in a separate list at the POS for quick access.

1. Open **Discounts**.
2. Click the desired discount.
3. Enable the **Favorite** checkbox.
4. Click **Save**.

> **Note:** Pay attention to the discount type when marking favorites. Different types appear in different lists at the POS.

---

## Frequently asked questions

**Can I print a discount as a QR code?**
Yes. Create a discount with a code and print it as a QR code label via the label printer.

**Can I limit a discount to a time period?**
Yes. Fill in the **Active from** and **Active to** fields.

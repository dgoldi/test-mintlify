You set up the bank connection and activate the "Invoice" payment method so the system automatically creates a Swiss QR code on every invoice with a Swiss address.

---

## Prerequisites

- Access to the **Admin App** with **Admin rights**
- A valid **bank connection** is set up in the system

---

## Set up QR invoices

1. Open the **Admin App**.
2. Navigate to **Bank Details** (`admin.subsidia.ch/bank-details`).
3. Open the desired bank connection.

![Bank details](/images/en/bankverbindungen.png)

4. Navigate to **Settings > Shifts** and select the desired shift.
5. Switch to the **Payment** tab.
6. Click **Add Payment Method**.
7. Select the type **Invoice**.
8. Select the corresponding **bank connection**.
9. Enter the **payment deadline**.
10. Click **Save**.

The system now automatically creates a Swiss QR code on every invoice with a Swiss address.

---

## Frequently asked questions

**Does the QR code appear on all invoices?**
The QR code only appears on invoices with a Swiss address.

**Do I need a special bank connection?**
The bank connection must be set up in the system. The QR code is generated from the bank details and invoice information.

---

## Prerequisites

- Access to the **Admin App** with **Admin rights**
- Permission for the **Settings** section

---

## 1. Create a new user

1. Open the **Admin App**.
2. Click **Settings** in the sidebar.
3. Click **Users**.
4. Click **Add User** in the top right.

![User list with Add User button](/images/en/benutzer-liste.png)

5. Fill in the following fields:

| Field | Required | Description |
|-------|----------|-------------|
| **Name** | Yes | First and last name of the user. |
| **E-Mail** | Yes | Email address for login. Must be unique. |
| **Role** | Yes | User permissions (see role descriptions below). |
| **Main Store** | No | Optional assignment to a store. |

6. Click **Save**.

![User form with fields](/images/en/benutzer-formular.png)

The system creates the user and sends an email for password setup.

---

## 2. Assign roles

Each user requires at least one role:

| Role | Description |
|------|-------------|
| **Admin** | Full access to the Admin App. Can manage users, settings, and all sections. |
| **Office** | Access to the Office App. Can handle stock movements and daily operations. |
| **POS** | Access to the POS App. Can process sales transactions. |

A user can have multiple roles at the same time.

---

## 3. Create a team account

Team accounts allow multiple people to share one account on a common device.

1. Create a new user as described in step 1.
2. Enable the **Team Account** checkbox.
3. Assign the desired roles.
4. Click **Save**.

> **Note:** Each team account requires a unique email address. This can be a pseudo address. Team accounts cannot manage users — this requires an Admin account.

---

## 4. Edit a user

1. Open **Settings > Users**.
2. Click the desired user.
3. Adjust name, roles, or main store.
4. Click **Save**.

---

## 5. Delete a user

1. Open the desired user.
2. Click **Delete** in the top right.
3. Confirm the deletion.

---

## Frequently asked questions

**Can a user have multiple roles?**
Yes. You can assign the Admin, Office, and POS roles to a user simultaneously.

**What is the difference between a user and a team account?**
A user is a personal account. A team account is a shared account for multiple people on one device. Team accounts cannot manage settings or users.

Sie erstellen Rabatte mit verschiedenen Typen und Einschränkungen und markieren häufig verwendete Rabatte als Favoriten.

---

## Voraussetzungen

- Zugang zur **Admin-App** mit **Admin-Rechten**
- Berechtigung für den Bereich **Rabatte**

---

## 1. Neuen Rabatt erstellen

1. Öffnen Sie die **Admin-App**.
2. Klicken Sie in der Seitenleiste auf **Rabatte**.
3. Klicken Sie auf **Rabatt hinzufügen**.

![Rabattliste mit Schaltfläche Rabatt hinzufügen](/images/de/rabatte-liste.png)

4. Füllen Sie die folgenden Felder aus:

| Feld | Beschreibung |
|------|--------------|
| **Name** | Bezeichnung des Rabatts. |
| **Typ** | Art des Rabatts (siehe Rabatttypen unten). |
| **Prozent** oder **Betrag** | Rabattwert in Prozent oder als fester Betrag. |
| **Code** | Optionaler Barcode für den Rabatt. |
| **Einmalig verwendbar** | Wenn aktiviert, kann der Rabatt nur einmal eingelöst werden. |
| **Aktiv von / bis** | Optionaler Gültigkeitszeitraum. |
| **Kunde** | Optionale Zuordnung zu einem bestimmten Kunden. |
| **Einlösbare Filialen** | Optionale Einschränkung auf bestimmte Filialen. |

5. Klicken Sie auf **Speichern**.

![Rabatt-Formular mit Feldern](/images/de/rabatt-formular.png)

---

## 2. Rabatttypen

| Typ | Beschreibung |
|-----|--------------|
| **Artikel-Rabatt** | Wird auf den zuletzt gescannten Artikel angewendet. Wiederholbar pro Artikel. |
| **Neuer Preis** | Setzt einen neuen Preis auf den zuletzt gescannten Artikel. |
| **Transaktions-Rabatt** | Wird am Ende der Transaktion angewendet. Nicht auf einzelne Positionen verteilt. |
| **Rabatt verteilen** | Wird auf alle Positionen verteilt. Passt sich automatisch an, wenn neue Artikel hinzugefügt werden. |
| **Individueller Artikel-Rabatt** | Wird einmalig auf alle Positionen angewendet. Kann nachträglich pro Artikel angepasst werden. |

---

## 3. Rabatt als Favorit markieren

Favoriten erscheinen in einer separaten Liste am POS für schnellen Zugriff.

1. Öffnen Sie **Rabatte**.
2. Klicken Sie auf den gewünschten Rabatt.
3. Aktivieren Sie die **Favorit**-Checkbox.
4. Klicken Sie auf **Speichern**.

> **Hinweis:** Achten Sie auf den Rabatttyp, wenn Sie Favoriten markieren. Verschiedene Typen erscheinen in unterschiedlichen Listen am POS.

---

## Häufige Fragen

**Kann ich einen Rabatt als QR-Code drucken?**
Ja. Erstellen Sie einen Rabatt mit einem Code und drucken Sie diesen als QR-Code-Etikett über den Etikettendrucker.

**Kann ich einen Rabatt zeitlich begrenzen?**
Ja. Füllen Sie die Felder **Aktiv von** und **Aktiv bis** aus.

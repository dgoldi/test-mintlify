Sie erfassen neue Benutzer, weisen Rollen zu und erstellen Team-Accounts für den gemeinsamen Zugang an einem Gerät.

---

## Voraussetzungen

- Zugang zur **Admin-App** mit **Admin-Rechten**
- Berechtigung für den Bereich **Einstellungen**

---

## 1. Neuen Benutzer erfassen

1. Öffnen Sie die **Admin-App**.
2. Klicken Sie in der Seitenleiste auf **Einstellungen**.
3. Klicken Sie auf **Benutzer**.
4. Klicken Sie oben rechts auf **Benutzer erfassen**.

![Benutzerliste mit Schaltfläche Benutzer erfassen](/images/de/benutzer-liste.png)

5. Füllen Sie die folgenden Felder aus:

| Feld | Pflicht | Beschreibung |
|------|---------|--------------|
| **Name** | Ja | Vor- und Nachname des Benutzers. |
| **E-Mail** | Ja | E-Mail-Adresse für die Anmeldung. Muss eindeutig sein. |
| **Rolle** | Ja | Berechtigungen des Benutzers (siehe Rollenbeschreibung unten). |
| **Haupt-Filiale** | Nein | Optionale Zuordnung zu einer Filiale. |

6. Klicken Sie auf **Speichern**.

![Benutzerformular mit Feldern](/images/de/benutzer-formular.png)

Das System erstellt den Benutzer und sendet eine E-Mail zur Passwort-Einrichtung.

---

## 2. Rollen zuweisen

Jeder Benutzer benötigt mindestens eine Rolle:

| Rolle | Beschreibung |
|-------|--------------|
| **Admin** | Vollzugriff auf die Admin-App. Kann Benutzer, Einstellungen und alle Bereiche verwalten. |
| **Office** | Zugriff auf die Office-App. Kann Warenbewegungen und Tagesgeschäft bearbeiten. |
| **POS** | Zugriff auf die POS-App. Kann Verkaufstransaktionen durchführen. |

Ein Benutzer kann mehrere Rollen gleichzeitig haben.

---

## 3. Team-Account erstellen

Team-Accounts ermöglichen mehreren Personen, sich ein Konto an einem gemeinsamen Gerät zu teilen.

1. Erstellen Sie einen neuen Benutzer wie in Schritt 1 beschrieben.
2. Aktivieren Sie die Checkbox **Team Account**.
3. Weisen Sie die gewünschten Rollen zu.
4. Klicken Sie auf **Speichern**.

> **Hinweis:** Jeder Team-Account benötigt eine eindeutige E-Mail-Adresse. Diese kann eine Pseudo-Adresse sein. Team-Accounts können keine Benutzer verwalten — dafür ist ein Admin-Account erforderlich.

---

## 4. Benutzer bearbeiten

1. Öffnen Sie **Einstellungen > Benutzer**.
2. Klicken Sie auf den gewünschten Benutzer.
3. Passen Sie Name, Rollen oder Haupt-Filiale an.
4. Klicken Sie auf **Speichern**.

---

## 5. Benutzer löschen

1. Öffnen Sie den gewünschten Benutzer.
2. Klicken Sie oben rechts auf **Löschen**.
3. Bestätigen Sie die Löschung.

---

## Häufige Fragen

**Kann ein Benutzer mehrere Rollen haben?**
Ja. Sie können einem Benutzer die Rollen Admin, Office und POS gleichzeitig zuweisen.

**Was ist der Unterschied zwischen Benutzer und Team-Account?**
Ein Benutzer ist ein persönlicher Account. Ein Team-Account ist ein gemeinsamer Account für mehrere Personen an einem Gerät. Team-Accounts können keine Einstellungen oder Benutzer verwalten.
